# 《今夜写点啥》Web 端 v3 重构计划

## 目标

把 `output/prototypes/night-writing-v3-prototype.png` 的方向落到正式 Web 页面：夜间可写光域、移动端 App 式历史和清空确认、输入时唯一的呼吸动效。

## 范围

- 只改 Web 端。
- 不改小程序。
- 不改历史导出、关键词算法、存储结构等非视觉核心逻辑。
- 正式页面涉及文件预计为：`index.html`、`styles.css`、`app.js`、`tests/history-hidden.test.js`。

## 任务清单

### 1. 先补测试约束

- 更新 `tests/history-hidden.test.js`。
- 验证 CSS 缓存版本号更新。
- 验证页面不再使用 `本机保存 · 不登录 · 不上传` 这种中圆点串。
- 验证存在清空确认底部面板结构。
- 验证 CSS 有写作光域变量、底部抽屉把手、动效和 `prefers-reduced-motion` 兜底。
- 验证 JS 有清空确认面板状态控制。

### 2. 重构首屏光域

- 主要修改 `styles.css`，少量修改 `index.html`。
- 把当前星点背景推进成 v3 的夜间可写光域。
- 去掉硬纸片感，保留斜向灯光、中心聚焦、边缘衰减。
- 桌面端正文限制到舒适宽度，让文字落在光域里。
- 关键词退到背景，不能抢正文。

### 3. 实现唯一标志性动效

- 修改 `styles.css` 和 `app.js`。
- `setVisualState()` 输出新的 CSS 变量，让输入量驱动低光、微亮、收束。
- 写作中只让光域轻微呼吸。
- 写够后保留“晚安”，但弱化为收束状态。
- `prefers-reduced-motion` 下全部变静态。

### 4. 移动端 App 化

- 修改 `index.html`、`styles.css`、`app.js`。
- 历史面板继续使用底部抽屉，加顶部把手和更明确层级。
- 清空确认从“按钮文字变成再点确认”改成底部确认面板：`保留 / 清空`。
- 保存状态保持轻量提示，不挡正文和底部面板。
- 保证按钮触控高度至少 44px。

### 5. 本地验收

- 运行 `node --test tests/history-hidden.test.js`。
- 运行 `git diff --check`。
- 用浏览器检查这些状态：
  - 桌面初始
  - 桌面输入
  - 桌面晚安
  - 移动输入
  - 移动历史抽屉
  - 移动清空确认
- 检查控制台无错误。

## 发布说明

本计划只要求完成本地 Web 重构。若要发布线上，再按项目规则同步 `main` 和 `gh-pages`，并验证 GitHub Pages 真实 URL。
